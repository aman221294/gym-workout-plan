## 🎯 Purpose

- Sell workout plans and motivate users to join fitness programs.
- Provide a detailed weekly workout plan (Chest, Back, Shoulders, Biceps, Triceps, Legs+Core).
- Demonstrate modern UI/UX with smooth animations, hover effects, and glowing elements.

---

## ✨ Features

- **Hero section** with gym background, animated text, and call-to-action button.
- **6 workout days** (Monday to Saturday) – each day includes:
  - Day name & muscle group
  - 5–8 exercises (name, sets, reps)
  - High-quality Unsplash image
  - Short description
- **Card layout** with hover animations (scale, neon glow).
- **Fully responsive** – works on mobile, tablet, and desktop.
- **JavaScript dynamic rendering** – workout data is injected via JS for easy updates.
- **Smooth scroll reveal animations** (fade-up effect).
- **Neon button effects** and interactive purchase plan popups.
